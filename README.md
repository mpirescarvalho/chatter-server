## Building

You'll need [Node.js](https://nodejs.org) installed on your computer in order to build this app.

```bash
$ git clone https://github.com/mpirescarvalho/chatter-server.git
$ cd chatter-server
$ npm install
$ npm run start
```

Runs the server in the development mode.<br/>
